# 🎁 願望顯示效果說明

## 新的模糊顯示功能

### 效果描述

在後台管理介面中，參與者的願望內容現在會以**模糊效果**顯示，而不是完全隱藏：

#### 模糊狀態（預設）
- 文字會被模糊處理（blur 效果）
- 背景有漸層閃爍動畫
- 文字顏色變透明，只能看到模糊的陰影
- 滑鼠懸停時模糊度會稍微降低（提示可以點擊）

#### 清晰狀態（點擊後）
- 文字完全清晰可讀
- 無模糊效果
- 正常的文字顏色

### 如何切換顯示

有兩種方式可以切換願望的顯示狀態：

1. **點擊表格標題**
   - 點擊「✨ 許願內容 (點我切換)」欄位標題
   - 會切換所有願望的顯示狀態

2. **點擊按鈕**
   - 點擊下方的「👀 偷看願望」按鈕
   - 按鈕文字會變成「🙈 把願望藏起來」

### CSS 效果細節

```css
/* 模糊狀態 */
.wish-content.hidden {
    color: transparent;              /* 文字透明 */
    text-shadow: 0 0 12px rgba(0,0,0,0.8);  /* 模糊陰影 */
    filter: blur(8px);               /* 模糊濾鏡 */
    background: 漸層背景;             /* 閃爍背景 */
    animation: shimmer 2s infinite;  /* 閃爍動畫 */
}

/* 懸停時 */
.wish-content.hidden:hover {
    filter: blur(6px);               /* 稍微降低模糊度 */
}
```

### 使用場景

這個設計的好處：

1. **保護隱私**：預設模糊，避免不小心看到願望
2. **視覺提示**：模糊效果讓人知道這裡有內容，但需要主動查看
3. **互動性**：閃爍動畫和懸停效果提示可以點擊
4. **童趣感**：符合聖誕主題的神秘感

## 測試步驟

1. 登入後台管理介面
2. 進入任一活動的詳情頁面
3. 觀察「許願內容」欄位：
   - 應該看到模糊的閃爍效果
   - 滑鼠懸停時模糊度會降低
4. 點擊表格標題或按鈕
5. 願望內容變清晰
6. 再次點擊，恢復模糊狀態

## 技術實現

- 使用 CSS `filter: blur()` 實現模糊效果
- 使用 `text-shadow` 增強模糊感
- 使用 CSS 動畫實現閃爍效果
- JavaScript 切換 `.hidden` class
- 平滑的過渡動畫（transition）
