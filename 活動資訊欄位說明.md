# 📋 活動資訊欄位說明

## 新增的欄位

在建立活動時，現在可以填寫以下額外資訊：

### 1. 📅 活動時間 (eventDate)
- **用途**：告訴參與者什麼時候要交換禮物
- **範例**：`2025/12/25 晚上 7:00`
- **顯示位置**：前台使用者頁面的活動資訊區塊
- **是否必填**：否（選填）

### 2. 📍 活動地點 (eventLocation)
- **用途**：告訴參與者在哪裡舉辦活動
- **範例**：`台北市信義區某某餐廳`
- **顯示位置**：前台使用者頁面的活動資訊區塊
- **是否必填**：否（選填）

### 3. 📝 備註說明 (eventNotes)
- **用途**：提醒參與者注意事項
- **範例**：`請準時到場，記得帶禮物喔！穿著輕鬆即可～`
- **顯示位置**：前台使用者頁面的活動資訊區塊（特殊樣式）
- **是否必填**：否（選填）

## 前台顯示效果

### 視覺設計

活動資訊會顯示在標題下方，使用紫色漸層背景的卡片：

```
┌─────────────────────────────────────────┐
│  🎄 2025 超級歡樂聖誕交換禮物大會       │
├─────────────────────────────────────────┤
│  💰 禮物金額： $500 - $700 元           │
│  📅 活動時間： 2025/12/25 晚上 7:00     │
│  📍 活動地點： 台北市信義區某某餐廳     │
│  📝 溫馨提醒：                          │
│     ┌───────────────────────────────┐   │
│     │ 請準時到場，記得帶禮物喔！    │   │
│     │ 穿著輕鬆即可～                │   │
│     └───────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### CSS 樣式特點

- **背景**：紫色漸層 (#e0e7ff → #ddd6fe)
- **邊框**：2px 實線，顏色 #c4b5fd
- **圓角**：15px
- **陰影**：柔和的紫色陰影
- **字體大小**：0.95rem（比正文稍小）
- **備註區塊**：白色半透明背景，左側紫色邊框，斜體字

### 圖示說明

- 💰 = 禮物金額
- 📅 = 活動時間
- 📍 = 活動地點
- 📝 = 溫馨提醒

## 注意事項

1. **所有新欄位都是選填的**，不填寫也不會影響系統運作
2. **如果沒有填寫任何資訊**，活動資訊區塊不會顯示
3. **只顯示有填寫的欄位**，例如只填寫時間和地點，就只顯示這兩項
4. **備註說明支援多行文字**，會以特殊樣式顯示
5. **文字大小比正文稍小**（0.95rem），不會太搶眼
